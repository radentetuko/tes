Siapa Aku?

Perkenalkan, saya Tetuko Suryo Guritno biasa dipanggil Koko, anak ke-2 dari dua bersaudara yang lahir pada tanggal 17 Juli 2001. Saya tinggal bersama keluarga kecil saya di Kabupaten Jember, Provinsi Jawa Timur, Indonesia. Ayah saya adalah seorang guru di suatu Sekolah Dasar di desa saya. Sedangkan Ibu saya adalah ibur rumah tangga, tetapi ibu juga membuka usaha menjahit baju. Selain menjadi guru ayah saya juga seorang petani setiap hari sepulang mengajar atau hari weekend ayah saya selalu menyempatkan bekerja di sawah, dan Saya selalu ikut untuk mendampinginya bekerja di sawah. Tetapi jika ada jam kuliah kuliah saya terpaksa untuk meninggalkannya bekerja sendiri. Saat ini saya sedang menempuh pendidikan di Universitas Negeri Jember jurusan Informatika. Singkat cerita dulu saya asal - asalan memilih jurusan itu, saya kira dulu jurusan Informatika adalah jurusan yang bekerja dibidang perangkat keras komputer tapi ternyata setelah masuk kuliah yang dipelajari adalah perangkat lunaknya. Jadi selama 3 semester saya berusaha dengan keras untuk beradaptasi dengan itu. Dan akhirnya saya menemukan jalan keluarnya di semester 4, di semester 4 saya menempuh mata kuliah pemrograman web, dimana mata kuliah tersebut mempelajari bagaimana cara kita membuat website dengan baik dan menarik, akhirnya saya jatuh cinta dengan pemrograman web dan saat ini di semester 5 saya mendalami pemgrograman web ini melalui program Kampus Merdeka. Di Program Kampus Merdeka ini saya mengambil track Front-End Web Development milik Mitra Skilvul.
